# Account-Management-System-using-Python
A beginner-friendly Python project that simulates an ATM account system using classes, private variables, and basic methods for viewing account details, checking balance, and depositing money.
This project is a basic Account Management System built in Python using Object-Oriented Programming (OOP). It demonstrates the use of private variables and class methods to manage account information securely. The system stores the account holder’s name and balance, displays user details, shows the current balance, and allows money to be deposited into the account.
