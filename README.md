# 🏦 Account Management System

## 📌 Description

This is a simple **Account Management System** built using Python.
It simulates basic ATM operations using Object-Oriented Programming (OOP) concepts like classes and private variables.

---

## 🚀 Features

* Display account holder name
* Check account balance
* Deposit money into account
* Uses private variable for security (`__balance`)

---

## 🛠️ Technologies Used

* Python
* Object-Oriented Programming (OOP)

---

## 💻 How to Run

### 👉 Using Google Colab

1. Open the `.ipynb` file in Google Colab
2. Run all cells

### 👉 Using Python File

1. Download the `.py` file
2. Run in terminal:

```
python bank_account.py
```
